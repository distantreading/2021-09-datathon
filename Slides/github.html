What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

Step 1: Create a GitHub account

  • https://github.com/

Step 2: Install GitHub

  • https://git-scm.com/downloads
  • 2a. Commandline
  • 2b. GUI Client, e.g. GitHub Desktop https://desktop.github.com/

Step 2a: Use GitHub on the Commandline

  • Configure your username and email
    • git config --global user.name "Martina Scholger"
    • git config --global user.name "Martina Scholger"
  • Open datathon repository on GitHub and clone it
    • https://github.com/distantreading/2021-09-datathon
    • git clone https://github.com/distantreading/2021-09-datathon.git

Step 2a: Use GitHub on the Commandline

  • Add your file to the repository
    • Update the local version of a repository from a remote repository

      git pull origin main
    • Checking the status of the repository

      git status
    • Save your file to the folder Work/Texts/
    • Add your file to the git staging area

      git add mytext.xml
    • Provide a commit message

      git commit -m "My first commit"
    • Pushing commits to a remote repository

      git push origin main

Step 2b: GUI Client fo GitHub

  • Install GitHub Desktop
  • File > Clone respository
  • Save your file to the folder Work/Texts/
  • Make sure your local repository is up to date

    Fetch origin
  • Provide a summary and "Commit to main"
