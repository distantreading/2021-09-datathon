PDF of page images

  • First unpack the PDF using e.g. pdftohtml myPDF.pdf. This will (rather slowly) produce lots of graphic files, one per page. Put them all in a folder together.
  • Use tesseract to do OCR on each page e.g. like this 
    cd myPDF-folder; for f in *.png ; do tesseract $f `basename $f .png` -l XXX; done
    replacing XXX with the ISO 3 letter code for your language (e.g. bos for Bosnian)

Plain text

  • First check for characters which will upset the XML processor. Ampersands must be changed to &amp first; and pointy brackets to &lt;. All characters should be in Unicode.
  • Make the document well formed by wrap the whole thing in a single div type=”chapter” containing one monster paragraph
  • break up individual paragraphs by adding </p><p> at the start of each new one
  • break up into multiple divs by adding </div><div type=”chapter”> at the start of each new one

In most cases, you should be able to use the oXygen find-and-replace command to fix up a lot of the text. For example,

If you have...regexp find and replaceeffect
blank lines between paragraphswill introduce <p> tags in the right place
tags like this <L 42> at the start of each new pagewill replace them with appropriate <lb> tags.
